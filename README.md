A aplicação consiste em um aluguel de carros de varios modelos.

# Como executar o projeto

## Mobile
Pré-requisitos: Expo

```bash
# clonar repositório
git clone https://github.com/Beluhga/Rentx.git

# entrar na pasta do projeto Mobile
cd Rentx

# instalar dependências
yarn install

# executar o projeto
expo start
yarn api
```
